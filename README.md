# Client Gateway

## Dev
1. Clone the repository:
   ```bash
   git clone git@github.com:Nest-Microservices-courseK/client-gateway.git
   ```
2. ```bash
   cd client-gateway
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Create `.env` file based on `.env.example`
5. Run the project:
   ```bash
   npm run start:dev
   ```
